## 📖 Descripción

**Más de 1,300 programas educativos en Python** para el aprendizaje de:

- 🏛️ **Latín** - Declinaciones, vocabulario y gramática
- 🇬🇧 **Inglés** - Gramática, preguntas y conectores
- 📖 **Español** - Sintaxis, ortografía y morfología
- 🎮 **Juegos Educativos** - Aprendizaje interactivo

> **Novedad:** Menú principal unificado `./menu_principal.sh`
